Добро пожаловать в «Бессонный Чат»

Локальный мессенджер для волонтеров Бессонницы. Работает без интернета!

1. Скачайте приложение Element или используйте версию для браузера (см. п. 3)

Выберите приложение для своего устройства (iOS, увы, в пролете, пробуйте браузерную версию):

Android Windows macOS Linux (x86-64) Linux (arm)

На Android включите установку из неизвестных источников и откройте файл element.apk. На Windows и macOS установите Element как обычную программу.

2. Подключить приложение к серверу

Во всех приложениях используйте этот адрес сервера (homeserver):

Адрес сервера (homeserver URL): http://synapse.besson.chat

Android / iOS (Element Mobile)

  1. Откройте приложение Element.
  2. Выберите «Создать аккаунт» или «Войти».
  3. Нажмите «Другой сервер» / «Custom server».
  4. В поле сервера введите: http://synapse.besson.chat.
  5. Продолжите регистрацию или вход по инструкции в приложении.

Windows / macOS (Element Desktop)

  1. Запустите Element на компьютере.
  2. На экране входа выберите «Другой сервер» или «Use custom server».
  3. Укажите сервер: http://synapse.besson.chat.
  4. Зарегистрируйте новый аккаунт или войдите в существующий.

3. Использовать веб-версию (без установки)

Можно пользоваться чатом прямо из браузера на этом устройстве.

Открыть веб-версию Element

Если браузер пишет, что «десктопная версия не поддерживается», жмите кнопку «Continue Anyway.»

Устройство (телефон, ноутбук и т. д.) должно быть подключено к той же локальной сети, что и сервер besson.chat. Доступ в интернет не требуется.
